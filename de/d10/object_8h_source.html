98 /*
99  CObject is described in more detail in: Q:\sk4\Docs\programming\Object Scripting System.htm
100 
101  The base class for all scriptable objects. Every CObject has a global ID that
102  is unique among all CObjects. It also has a type specifier.
103 
104  A script that is running can reference a CObject. The script can access special
105  commands for that CObject through CallMemberFunction(). Tags, which are like
106  scriptable member variables, can be attached from CObjects from script, too.
107 
108  Event handling is supported through the PassTargetedEvent() function.
109 */
