132 // Script class.
133 // To run a script, one must create one of these, then call the SetScript member function to
134 // set which script it is to run.
135 //
136 // Then the Update function must be called repeatedly.
137 // The script will never die by itself, it must be deleted.

139 // Poolable to speed up allocation & deallocation.

142 // of CScript's ever gets big enough to be using up a substantial amount of memory.
143 // There would be a slight speed hit if pools were used, but probably not much,
144 // allocating/deallocating off a pool is pretty quick.
